##Requisites
 * nodejs >= 8.11.3
 * mongodb >= 4.0.0

##Installation
Install dependencies with:

```shell
npm install
```

###MongoDB
Install MongoDB with: 
````
brew install mongodb
```

start mongodb services with:

finally run command:
```shell
npm run installDB
```
